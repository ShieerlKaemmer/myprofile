Project Ideas

For this project, I would say develop an application, especially gaming application is the most interesting out of all when I first saw the project idea lists. It can be any game categories such as action, educational, simulation and etc. I would like to try how to make a game about escape room as I love to play escape room when I was in my hometown, but it is expensive for me to play. Therefore, I thought it would be a good idea to make an escape room simulation game.

As I mention that mobile application developer is my ideal job, it would be really interesting for me to learn how to develop a gaming app using any programming languages or tools such as unity, Construct2 or Cocos2d as it will give me the experience I need when I have to develop a application, even though it is a game application but I thought it still might be a useful experience for me. According to an article in GeekWire, there are around 1.2 billion people play games in this world and the users are mostly teenager, so it is easier for us to target the user, which is teenager who loves playing escape room.

As I mention before that I want to make an escape room game, it is a game specially design for people who likes to play escape room but does not want to pay too much for it. Even though it is just a simulation but since it is free, so I thought It would be a good idea to design it for them. The game will ask the user to find a way to escape the room by finding an object or hints inside the room that will lead the user to escape the room they are staying at the moment. The difficulties for the next level will increase as the user successfully escape from the previous room. When you can’t continue anymore as you have no idea what to do next, you are allowed to ask for help and we will give you a hint which might help you to find the key or objects you need, but of course the hint are limited as we still want user to figure out themselves, it is not fun when we spoil the answers at one time, so the user will have to wait 15 minutes to get the next hint and there is no hint available when you point is zero. How does user earn points? It is given every time when you finish one level, the less hint you ask, the more point will be given, but there is a minimum points given to the user every time they finish one level, so they do not have to worry about no hint given on the next level. As the difficulties increases, the points given will increases as well, since we do not want user to struggle to much when they are using our project, so they do not have to worry that only one or few hints will be given next level. The user will repeat the same way until they have reached the maximum level of our game.

The tool needed would be either Unity, construct2 or Cocos2d, it’s a tool which you can use it for creating a gaming application. I would choose Unity as it is more famous compare to construct2 or cocos2d, and I saw their description saying that there are more than 50% of mobile games are made from Unity so I thought it would be a good idea to use unity for this project. Unity has a really good reputation among game developer as I scroll through the reviews and feedbacks every game developer company gave them on their website. Thus, I would say using Unity will be easier for us as there are so many companies using it, so when we have a problem that have to fix, it might be easier for us to find the solutions.

Basically, there are 3 unity plans available on their website. There is a free version for beginner, 35$ per month for hobbyist and 125$ per months for team and freelancer. It depends for everyone needs, but I think the free version or the 35$ plan would be enough as we are using it for IT project, so I think 125$ in unnecessary for us since we are not a game developer company. We only have to download unity into our laptop so it will not take much time to get ready for the project.

If the project is successful, an escape room game application would be the outcome. People who needs to entertain themselves during free time can play our gaming application whenever they want. I believe that people who likes escape room game will love our game as it is free, and they can play anytime they want without someone has to tell them that they reach the time limit.
